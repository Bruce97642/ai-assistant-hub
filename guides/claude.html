Claude 網頁版與 App:文書工作的起點

claude.ai/手機 App/桌面版 — 對話即生產

最基本、也最被低估的入口。除了問答之外,網頁與 App 版的 Claude 能直接產出真正的檔案:Word(.docx)、Excel(.xlsx)、PowerPoint(.pptx)、PDF,做完直接下載,不是只給你一段文字要你自己貼。

它能替文書工作者做什麼

📄 文件生成

會議紀錄、簽呈、公文、企劃書、合約草稿、新聞稿——描述需求即可產出完整 Word 或 PDF 檔,含標題階層、目錄、表格與頁碼。

📊 表格與資料

貼上雜亂資料或上傳 CSV,Claude 幫你清理、去重、做樞紐分析、產出含公式與圖表的 Excel 檔。

🖥️ 簡報製作

「幫我把這份報告做成 12 頁的提案簡報」——直接生成 .pptx,可再逐頁要求修改。

🔍 文件分析

上傳 PDF、合約、報表、掃描檔,請 Claude 摘要重點、比對差異、抓出風險條款、翻譯成雙語對照。

📁 專案(Projects)

把常用背景資料(公司簡介、格式範本、術語表)放進專案,之後每次對話 Claude 都自動記得,不必重複交代。

⚙️ 技能(Skills)

把你的固定作業流程(例如「週報格式」「客戶信範本規則」)存成技能,一句話就套用,全組織可共用。

操作圖解:五步驟做出一份正式報告

  1. 開新對話,上傳原始素材點輸入框旁的「+」→ 上傳檔案。可一次丟入會議逐字稿、Excel 數據、舊版報告等多個檔案。
  2. 用一句話說清楚「產出物」例:「請根據這三個檔案,寫一份 10 頁的季度營運檢討報告,產出 Word 檔,格式比照上傳的舊版報告。」關鍵字:檔案格式、頁數、比照範本。
  3. Claude 分析 → 生成檔案Claude 會先讀完素材、規劃章節,再實際建立 .docx 檔。過程中可看到它的工作步驟。
  4. 逐項修改,不必重來直接說「第三章加一張同比成長表」「語氣改正式一點」,Claude 會在同一份檔案上修改。
  5. 下載交付點檔案卡片即可下載,直接寄出或再微調。
提 示 語 範 例「附件是本週五場會議的逐字稿。請整理成一份會議紀錄 Word 檔:每場會議一節,含出席者、決議事項、待辦(負責人+期限),最後加一頁跨會議待辦總表。」
小 技 巧要求「比照我上傳的範本格式」是文書工作最省力的一招——Claude 會沿用你的標題樣式、字體層級與表格版式,產出直接符合公司格式的檔案。

Claude for Chrome:住進瀏覽器的助理

Google Chrome 外掛(Beta)— 會點擊、會填表、會自己跑流程

安裝後,Claude 以側邊欄形式常駐在 Chrome 裡:它看得到你正在看的網頁,並且能在你授權下實際點擊按鈕、填寫表單、切換分頁、擷取資料。目前為 Beta 功能,開放給所有付費方案(Pro、Max、Team、Enterprise)。

文書工作者的五大殺手級用法

📝 自動填表

報帳系統、請購單、政府申辦網站、問卷後台——把資料給 Claude,它逐欄替你填好,你只需最後確認送出。

📧 信箱整理

請 Claude 掃描 Gmail,找出行銷信與通知信列成清單供你確認後批次刪除;或替你草擬回信。

🌐 跨網站彙整

把多個分頁拖進 Claude 的分頁群組,它能同時讀取所有分頁,比對競品價格、彙整多站資訊成一張表。

🎬 錄製工作流程

按下錄製鍵,自己示範一次操作步驟(例如每週上傳週報到內部系統),Claude 學會後即可重複代勞。

⏰ 排程任務

點外掛面板右上角的時鐘圖示,可將流程設為每日/每週/每月自動執行——例如每週一早上自動彙整上週表單回覆。

🧭 熟門熟路

Claude 內建對 Gmail、Google 日曆、Google Docs、Slack、GitHub 等常用平台的操作知識,交辦時不必教它介面。

操作圖解:從安裝到第一個任務

  1. 安裝外掛到 Chrome 線上應用程式商店搜尋「Claude」→ 加到 Chrome →(建議)點拼圖圖示把 Claude 釘選到工具列。僅支援 Chrome 桌面版。
  2. 登入 Claude 帳號使用你既有的付費帳號登入,並授予外掛所需權限。
  3. 選擇權限模式建議新手用「先詢問再行動」:Claude 會先提出計畫給你核准,再在核准範圍內自主執行;高風險動作(發佈、購買、送出個資)一律會再問你。
  4. 打開側邊欄,交辦任務點工具列的 Claude 圖示 → 側邊欄開啟 → 用白話描述:「幫我把這頁表單用右邊這份資料填好,填完先讓我檢查。」
  5. 確認 → 執行 → 驗收看著 Claude 即時操作。重複性的流程可進一步「錄製」或「排程」,之後一鍵重跑。
提 示 語 範 例「打開這三個競品官網的分頁我已放進你的分頁群組。請比對三家的方案價格與功能差異,整理成一張比較表,並標出我們最有優勢的三點。」
安 全 提 醒瀏覽器 AI 可能遭遇「提示注入」(網頁中隱藏的惡意指令)。請從熟悉、信任的網站開始授權;金融交易、密碼管理、敏感個資請勿交給外掛處理;Claude 對高風險網站(如銀行)會自動停用。逐站授權可隨時在設定中收回。

Claude for Microsoft 365:辦公室四大金剛

Excel・PowerPoint・Word 正式版/Outlook Beta — 側邊欄直接改你的檔案

從 Microsoft 市集(AppSource)安裝增益集後,Claude 會以側邊欄出現在 Excel、PowerPoint、Word 與 Outlook 裡,直接讀取並修改你當前打開的檔案。最強大的是「跨應用程式脈絡」:在 Excel 分析完的數據,切到 PowerPoint 時 Claude 還記得,不必重新解釋、不必複製貼上。

四個應用程式,各自能做什麼

Excel|表格與模型

用白話描述就能寫公式、除錯公式;稽核財務模型(抓循環參照、該用公式卻寫死的數字);清理資料、去重、補漏;依範本自動填入 DCF/LBO 等模型。所有變更會標示出來供你檢查。

PowerPoint|簡報

依你的公司範本與投影片母片生成新頁面、改寫選取的文字、產生原生圖表與圖解、統一全簡報的語氣與版式。可在「Instructions」欄設定品牌規範,之後每次都自動套用。

Word|文件

起草、改寫、摘要、依公司樣式更新內容;所有修改以 Word 原生「追蹤修訂」呈現,你逐條接受或拒絕,像審同事的稿一樣透明;還能回覆文件內的註解串。

Outlook|郵件(Beta)

一句話分類整理收件匣、摘要冗長信件串、抓出待辦事項、草擬回信與會議邀請——但任何信件都要你按下送出才會寄出。

圖解:一條龍跨應用工作流(這就是它的精髓)

OUTLOOK早上先問:「今天哪些信需要我處理?」Claude 分類收件匣、替幾封簡單的擬好回信,並從客戶來信中打開附件簡報需求。
WORD「依這封信的需求,用公司範本起草提案備忘錄」信件脈絡自動帶入 Word,Claude 以追蹤修訂方式產出草稿。
EXCEL「替備忘錄裡的方案建一個三年財務試算」Claude 已理解備忘錄內容,直接在 Excel 建模型;你調整假設,公式不會被弄壞。
POWERPOINT「把以上分析做成 8 頁決策簡報,套用公司範本」數據、結論、圖表自動從 Excel 與 Word 帶入簡報——全程零複製貼上。

操作圖解:安裝與啟用

  1. 安裝增益集在任一 Office 應用程式中:插入 → 取得增益集 → 搜尋「Claude by Anthropic」→ 安裝(或由公司 IT 從 Microsoft 365 管理中心統一佈署)。
  2. 啟用並登入Windows:常用 → 增益集;Mac:工具 → 增益集。開啟 Claude 面板後以 Claude 帳號登入。
  3. (選用)開啟跨檔案功能在各增益集的 Settings 開啟「Let Claude work across files」,並確保要協作的檔案都已打開——Claude 只能讀寫「目前開啟」的檔案。
  4. 設定 Instructions 常駐指示例如在 Excel 寫「輸入格用藍字、公式用黑字」,在 PPT 寫「條列每行不超過一行」,之後每次對話自動遵守。
  5. 交辦、檢查、接受Excel 會醒目標示變更、Word 走追蹤修訂——養成「先檢查再存檔」的習慣即可放心使用。
提 示 語 範 例(Excel)「稽核這個模型:找出循環參照、寫死的數字、加總不平衡的地方,列成清單並說明建議修法,先不要直接改。」
團 隊 加 乘常用流程可存成「技能(Skills)」全組織共享:合約審查清單、週報格式、簡報品牌規範……新人也能一鍵產出符合團隊標準的成果。Excel、PPT、Word 已對所有付費方案正式開放,Outlook 為 Beta。

Claude Cowork:你的桌面數位同事

桌面版代理 — 直接在你電腦的資料夾裡工作,交辦一次、拿回成品

Cowork 是為「不寫程式的知識工作者」設計的代理式工具:你指定一個工作資料夾並授權,Claude 就能直接讀取、編輯、建立本機檔案,在受控的隔離環境中把整件事做完——不必上傳、下載、也不必把工作拆成一句一句的指令。它的定位是:你給結果,它包過程。透過所有付費方案的 Claude 桌面應用程式即可使用。

最適合交給 Cowork 的四類文書工作

🗂️ 資料夾大掃除

下載資料夾裡混雜著草稿、附件、截圖?請它依專案分類、統一命名規則、揪出重複檔、把半年以上舊檔歸檔。

📑 多檔合成報告

丟給它一個裝滿來源檔的資料夾(訪談稿、數據、簡報),它自動閱讀、消化、產出結構完整的報告初稿——你只需做最後的判斷與修飾。

🔁 高工時重複作業

每月都要做的格式轉換、資料彙整、名單比對:描述一次流程,之後可設成排程任務定期自動執行。

🔎 掃描與盤點

那些「重要但一直沒空做」的事:掃過幾百筆客戶回饋找共通抱怨、盤點合約到期日、檢查文件版本差異——現在會真的被做完。

操作圖解:第一次使用 Cowork

  1. 開啟 Claude 桌面應用程式在 Mac 或 Windows 的 Claude 桌面版中切換到 Cowork。
  2. 指定工作資料夾並授權選一個資料夾作為 Claude 的工作範圍——它只碰你授權的地方,並在隔離的虛擬環境中運作。
  3. 描述「最終要拿到什麼」例:「把這個資料夾裡 37 份出差報告,彙整成一份 Excel 總表(人員/日期/金額/事由)加一份兩頁的費用分析摘要。」
  4. 核准計畫Claude 會先把任務拆解成步驟給你確認,重大決定仍由你拍板。
  5. 放手讓它跑,回來驗收成品期間你可以去做別的事(也能從手機 App 查看進度);完成後直接在資料夾裡拿到成品檔。
提 示 語 範 例「掃描『客服回饋』資料夾內所有檔案,把客訴依主題分類統計,產出:1) 分類統計 Excel;2) 前五大問題與代表性原文摘錄的 Word 報告;3) 給主管的十行重點信草稿。」
接 力 玩 法Cowork 可搭配 Chrome 外掛:讓 Chrome 上網逛競品網站蒐集價格與定位,Cowork 在桌面把一切彙編成格式完整的 PowerPoint——研究到成品之間不再有手動搬運。注意:代理任務消耗額度遠高於一般對話,Pro 方案較快達到用量上限。

Claude Code:進階者的自動化引擎

指令列/桌面/手機皆可驅動 — 批次、大量、可重現

Claude Code 原是給工程師的指令列代理,但 Anthropic 內部的行銷、數據等非技術團隊早就拿它處理複雜的多步驟工作——Cowork 正是由此誕生的簡化版。如果你的文書需求已經超過「一份一份做」,Code 就是下一站。

文書場景下的威力

⚡ 批次處理

「把這 200 份 Word 依新範本重排並轉成 PDF」「從 300 份合約抽出甲方、金額、期限做成總表」——一個指令,整批完成。

🧱 建立可重現流程

把「每月結報流程」寫成一次描述,之後每月同樣品質重跑;也可打包成技能(Skills)給團隊共用。

🛠️ 自建小工具

不會寫程式也能用白話請它做出「發票彙整器」「命名批次修正器」等專屬小工具,一勞永逸。

🌐 與 Chrome 聯手

Code 與 Chrome 外掛可整合:終端機處理檔案、瀏覽器上網驗證與抓取,打通線上線下。

操作圖解:非工程師的入門路徑

  1. 先從 Cowork 開始兩者底層能力相同;Cowork 介面友善、權限更受控,適合先建立手感。
  2. 需要批次與重現性時再升級安裝 Claude Code(終端機或桌面應用程式皆可),指向你的工作資料夾。
  3. 一樣用白話交辦不需要程式語言:「把 reports 資料夾裡所有 xlsx 的『台北』分頁彙總成一個總表」。
  4. 把常用流程存下來請它把成功的流程記錄成可重複執行的指令或技能,下次一句話重跑。
  5. 手機遠端交辦透過 Claude 手機 App 可遠端啟動與查看 Claude Code 任務——通勤時交辦,到公司收成品。
選 擇 心 法一句話記住三兄弟:聊天版是「問答與單件生產」、Cowork 是「桌面自動化」、Code 是「批次與工程級自動化」。多數文書工作者的日常,用前兩者就已足夠。

安全守則與使用心法

放手交辦,但把關永遠在你手上
✅ 人是最後一關

Word 走追蹤修訂、Excel 標示變更、Outlook 要你按送出、Cowork 先給計畫再執行——這些設計都是為了讓你審核。正式對外文件、法律與稽核文件,務必人工複核後再交付。

🔐 權限最小化

Chrome 外掛採逐站授權、可隨時收回;Cowork 只碰你指定的資料夾。原則:先小範圍試、確認可靠再擴大。

🚫 三不原則

不讓瀏覽器代理碰金融交易、不讓它管理密碼、不在未監督下處理高敏感個資。高風險動作 Claude 也會主動再次向你確認。

🧾 留下軌跡

Excel 可開啟 Claude Log 記錄每回合動作;Enterprise 方案可將 Office 增益集稽核紀錄接入企業自有監控系統,符合稽核需求。

代理式任務(Cowork/Code/Chrome 自動化)消耗額度遠高於一般聊天;把它們留給「高工時、高重複」的任務最划算。
